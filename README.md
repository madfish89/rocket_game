## About

Rocket Game is a lightweight browser game where the player controls a rocket and must avoid incoming obstacles for as long as possible. The game is implemented in JavaScript and HTML with a minimal, dependency-free setup — perfect for learning, demonstrations, or small arcade-style fun.

---

## Features

- Smooth, lightweight gameplay powered by vanilla JavaScript
- Increasing difficulty over time (speed / obstacle frequency)
- Simple scoring system to track survival time or points
- Easily extensible to add power-ups, levels, or new obstacle types

---

## How to Play

1. Start the game.
2. Keep the rocket afloat and maneuver to avoid obstacles.
3. Survive as long as possible to increase your score.
4. When you collide with an obstacle the game ends — try again to beat your best score!
