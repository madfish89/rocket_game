# Rocket Game


## About

Rocket Game is a lightweight browser game where the player controls a rocket to avoid incoming obstacles/asteroids with arrow keys and collect stars/white dots to up their score and level up. The game is implemented in JavaScript and HTML.
---

## Features
As you maneuver and level up in the game, the game sounds will change and the game will become more difficult with increased speed and obstacle spawning.
---

## How to Play

https://madfish89.github.io/rocket_game/
---

## Controls

- Arrow Up: thrust
- Left / Right arrow: Rotate rocket
- r: restart
